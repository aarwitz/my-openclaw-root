# People Profiles

Use this directory for durable, person-specific profiles that help with delegation, collaboration, and communication.

## Purpose

Keep stable information here:
- strengths
- weaknesses / constraints
- preferred task types
- communication style
- reliability patterns
- delegation guidance

Do **not** use these files for raw daily logs. Put new observations in `memory/YYYY-MM-DD.md`, then promote stable patterns into these profiles once they seem durable.

## Suggested Schema

- Name
- Relationship / role
- Core domains
- Strengths
- Weaknesses / watchouts
- Preferred tasks
- Avoid delegating
- Communication style
- Decision style
- Reliability / execution notes
- Delegation guidance
- Open questions
- Evidence / examples
- Last updated

## Update Rules

- Prefer concise bullets over long prose.
- Separate observed facts from tentative guesses.
- When possible, include evidence/examples for stronger claims.
- Update these files when a pattern appears stable over time.
